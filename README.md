# Mathieu Chateigner — Portfolio

Personal portfolio site, live at **[mathieu-chateigner.github.io](https://mathieu-chateigner.github.io/)**.

## Stack

- HTML / CSS / JavaScript (vanilla)
- Bootstrap 5 (local)
- Font Awesome 6 (local)
- AOS (Animate On Scroll, CDN)

No build step — served directly by GitHub Pages from the repository root.

## Structure

```
index.html              Main page
styles/style.css        Custom styles
scripts/main.js         Nav, scroll behaviour, AOS init
assets/
  cv-mathieu-chateigner.pdf
bootstrap/              Bootstrap 5 (local copy)
assets/fontawesome/     Font Awesome 6 (local copy)
```
